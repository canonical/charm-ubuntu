# Code of Conduct

This project follows the [Ubuntu Code of Conduct](https://ubuntu.com/community/ethos/code-of-conduct).

As contributors and maintainers of this project, we pledge to respect everyone who contributes by posting issues, updating documentation, submitting pull requests, and providing feedback.

If you believe someone is violating the code of conduct, please report it to the project maintainers or via [GitHub's reporting mechanism](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam).
